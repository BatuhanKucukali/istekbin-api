# Istekbin - Inspect HTTP Requests

Istekbin is a free service that allows you to collect http request.

## Run this project

1 . Clone project on your machine
```bash
git clone git@github.com:BatuhanKucukali/istekbin-api.git
```
2 . Change directory
```bash
cd istekbin-api
```
3 . Build
```bash
go build
```
4 . Run
```bash
./istekbin
```
